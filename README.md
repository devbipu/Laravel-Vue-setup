This is full stack Laravel & Vue.js Blog website project developed by Biplob Shaha. Admin panel & Frontend both are build with Vue js (spa) 





## Techonology used in the project
- Laravel 9.
- Vue 3.
- Vuex 4.
- Vue Router 4.
- iViewUi plus.


## Feature of the site

- Blog post with Category & Tags
- Comment & Reply
- Social Login
- User Profile
